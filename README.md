# BCG Cases Data Analysis

Analysis and prediction of barley yield in France using climate and geographical data.

## Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

## Required Setup

**Important**: The file `climate_data_from_1982.parquet` must be manually added to the `raw_data/` folder. This file is too large to be versioned on Git. Without this file, the notebooks will not run correctly.

## Project Structure

### Analysis Notebooks

- **`data_description_yield.ipynb`** : Descriptive analysis of barley yield data by French department (1982-2018). Temporal visualizations, statistical distributions, and geographical maps.

- **`model.ipynb`** : Construction of a barley yield prediction model using climate features (winter, stem elongation, and grain filling periods) and geographical features. Generates predictions for 2019-2050.

- **`map_barley_yield_2040.ipynb`** : Interactive cartographic visualization of barley yield predictions for the year 2040 by department.

### Data

- **`raw_data/`** : Folder containing raw data
  - `barley_yield_from_1982.csv` : Corrected barley yield data (generated by `data_correction.py`)
  - `climate_data_from_1982.parquet` : Climate data (to be added manually)
  - `original_data/barley_yield_from_1982.csv` : Original yield data
  - `data_correction.py` : Data correction script (adds department numbers, calculates missing yields)

- **`data_model_prediction/`** : Model-generated predictions
  - `predictions_yield_2019_2050.csv` : Yield predictions for 2019-2050, this file is generated from the 'model.ipynb' notebook

### Utilities

- **`utils/department_info.py`** : Module containing information about French departments (numbers, geographical coordinates, regions). Used by notebooks and the correction script.

## Workflow

1. Add the file `climate_data_from_1982.parquet` to `raw_data/`
2. Run `raw_data/data_correction.py` to generate corrected data (if necessary)
3. Run `data_description_yield.ipynb` for descriptive analysis
4. Run `model.ipynb` to train the model and generate predictions
5. Run `map_barley_yield_2040.ipynb` to visualize predictions on a map
